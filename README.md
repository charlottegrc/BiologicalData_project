# PF03060 Protein Family Analysis Pipeline

This project builds a workflow to analyze the Pfam family **PF03060**, from fetching sequences to building MSAs, PSSMs, and HMMs.

## Requirements

### Python Packages
- Python 3.9+
- Biopython
- Pandas
- NumPy

Install with:

```bash
pip install biopython pandas numpy
